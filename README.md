# DSCI-510-2025-Fall-JX3-Update-Analyze
# Project name: JX3 Updates Trends
## Team member: Yifan Zhang
## Project Description: This project analyzes historical update patterns of the MMORPG game JX3 (Jianwang 3). By scraping update information from the official JX3 website, the project studies how update sizes change over time and how frequently major updates occur. The goal is to identify patterns and trends in update sizes and update types.
## Data Source: Official JX3 Patch Website https://jx3.xoyo.com/patch/
